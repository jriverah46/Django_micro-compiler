Normas del Compilador

Este compilador soporta un conjunto de instrucciones básicas para la evaluación de expresiones aritméticas. Las normas son las siguientes:

  • Se pueden usar operadores matemáticos como +, -, *, /.
  • la instrucción para escribir en pantalla es : write (“ mensaje “, expresión )
  • la instrucción para capturar dato en pantalla es: capture(variable)
  • condicional: el condicional debe tener la siguiente sintaxis en minúscula - if (condición) then ……… ………. end-if
  • Las expresiones deben estar correctamente formateadas con paréntesis cuando sea necesario.
  • Se pueden usar operadores relacionales : <,>,<=,>=,=,<> y operadores lógicos : and , or , not
  • dentro de (condición) pueden haber más de una condición enlazadas con operadores lógicos y relacionales
  • Los números deben ser enteros y positivos.
  • Las instrucciones deben terminar con doble 2 puntos (::). esto no aplica a las líneas que tengas (if —- then) ni el (end-if)
  • las palabras reservadas deben colocarse de color “azul” o colocarse en “negrita” .
